# 🎬 YouTube Clone — Full Stack MERN

A full-featured YouTube clone built from scratch using the MERN stack.

## 🚀 Features
- User Authentication (JWT + Refresh Tokens)
- Video Upload with Cloudinary
- Like, Comment, Subscribe
- Watch History
- Protected Routes

## 🛠 Tech Stack
- **Frontend:** React, Vite, Redux Toolkit, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB + Mongoose
- **Auth:** JWT (Access + Refresh tokens)
- **Storage:** Cloudinary

## 📦 Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
